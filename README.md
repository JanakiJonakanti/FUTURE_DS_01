Business Sales Performance Analytics

Project Overview

This project analyzes business sales data to identify revenue trends, top-performing products, profitable categories, and high-performing regions. The analysis was performed using Power BI and provides actionable business insights through an interactive dashboard.

Objectives

- Analyze overall sales and profit performance
- Identify top-selling products
- Examine sales trends over time
- Compare category-wise performance
- Evaluate regional profitability
- Provide business recommendations for growth

Dataset

Dataset Used: Superstore Sales Dataset

Source:
https://www.kaggle.com/datasets/vivek468/superstore-dataset-final

The dataset contains:

- Order Date
- Product Name
- Category
- Region
- Sales
- Profit
- Quantity

Tools Used

- Microsoft Power BI
- Microsoft Excel (Data Cleaning)

Dashboard Features

KPI Cards

- Total Sales
- Total Profit
- Total Orders

Visualizations

- Sales Trend Analysis
- Top 10 Products by Sales
- Category-wise Sales and Profit
- Region-wise Performance Analysis
- Interactive Filters (Region, Category, Date)

Key Insights

- Technology category generated the highest revenue.
- West region contributed the highest profit.
- Phones were among the best-selling products.
- Sales showed steady growth over the analyzed period.

Recommendations

- Increase focus on high-performing product categories.
- Expand marketing efforts in profitable regions.
- Improve performance of low-selling products.
- Maintain inventory for top-selling products.
